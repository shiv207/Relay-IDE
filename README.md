## About Relay

Relay is a next-generation IDE built for developers who can't wait for their agents to finish writing code. Born from the frustration of watching progress bars while scrolling through Hinge, Relay is designed for the brainrot generation that demands speed, efficiency, and token optimization.

## Key Features

- **Advanced Token Optimization**: Cutting-edge token management to maximize your AI coding budget
- **Lightning-Fast Performance**: Built for developers who need results now
- **Modern Architecture**: Built on VS Code's open-source foundation with custom optimizations
- **Niche Market Focus**: Designed for developers who live on the edge of technology and dating apps

## Token Optimizations

Relay integrates advanced token optimization strategies including:
- Context-aware token compression
- Smart caching mechanisms
- Predictive token usage
- Multi-model token routing

## Getting Started

### Prerequisites
- Node.js (18 or higher)
- npm or yarn
- Git

### Installation

```bash
git clone https://github.com/shiv207/Relay---IDE-for-the-brainrot-generation.git
cd Relay---IDE-for-the-brainrot-generation
npm install
npm run build
npm run start
```

## Development

Relay is built on the VS Code open-source codebase with significant modifications for token optimization and performance enhancements.

### Building from Source

```bash
npm run build
```

### Running Tests

```bash
npm test
```
